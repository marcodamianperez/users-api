Users API

The simplest API ever

Try it

Overview

Users API is a free and fake API that provides AI-generated users data for when you are developing your app. Do not waste time creating a JSON file for testing purposes. Focus on the code.

Resources

Users API provides one type of resource.

/users 50 users

Routes

GET is the only HTTP method supported by Users API.

GET /users
GET /users/1

That's as simple as it gets. If you want to know more, please visit the GitHub repo.
